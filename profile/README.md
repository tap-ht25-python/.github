# Kodexempel
## Programmering med Python
Denna GitHub-organisation innehåller kodexempel från lektionerna i kursen.

För att se kodexempel:
+ klicka på fliken "Repositories"
+ klicka på det "repo" (repository) du är intresserad av
+ klicka på en fil för att se innehållet

Du kan också ladda ner koden för att se den enklare i din kodeditor.

### Ladda ner kod med git från terminalen
Detta kräver att du har installerat git. Kontrollera om du har git genom att skriva `git status` i valfri terminal. Om du får meddelandet att terminalen inte hittar kommandot git måste du installera det: [ladda ner git](https://git-scm.com/install/)

1. Klicka på repot du är intresserad av. Klicka på den gröna knappen `<> Code` och kopiera repots "web URL".
2. Öppna valfri terminal och navigera till en mapp du vill ladda ner koden till
3. Skriv `git clone` och klistra in web URL som du kopierade i steg 1
4. Om du gjort rätt kommer git ladda ner koden och lägga den i en mapp med samma namn som repot. Skriv `cd mappnamn` för att gå in i mappen. Skriv `ls` eller `dir` för att se att filerna finns där. Nu kan du öppna mappen i din kodeditor.

### Alternativ
Vi kommer använda PyCharm i kursen, men VS Code har också bra stöd för Python och Git. Du kan ladda ner koden inifrån båda editorerna.
Alla kodeditorer har inbyggt stöd för versionshantering. Börja med att kopiera web URL på samma sätt som i steg 1 ovan. Leta sedan upp rätt meny för att skriva in den. Editorn kommer att klona projektet åt dig.
